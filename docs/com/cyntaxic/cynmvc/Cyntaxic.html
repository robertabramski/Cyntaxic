Packagecom.cyntaxic.cynmvc
Classpublic class Cyntaxic
InheritanceCyntaxic Object

The Cyntaxic framework is a MVC based design pattern created to address everyday coding, while keeping it simple. Top level objects like root & stage can be referenced statically from the Cyntaxic base class. It also comes bundled with a bunch of commonly used utility functions and a built in debugger.

describe()method
public static function describe(object:Object, compact:Boolean = true):String

Returns a valid JSON string describing the object passed in the first argument. Objects that can't be used in JSON like the Function object are returned as a string literal. The controller and views have shortcut functions for this that assume the first argument as itself. Use this when the intention is to describe an object without the describe function.

Parameters

object:Object — If false the string is not compacted.
 
compact:Boolean (default = true)

Returns
String — A valid JSON string.

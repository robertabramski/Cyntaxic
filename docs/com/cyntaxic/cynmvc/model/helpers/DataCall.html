public function DataCall(handle:String, url:String, method:String = get, data:Object = null, contentType:String = text/plain)

Creates a new DataCall object. DataCall makes calls to an external source and returns the data.

Parameters
handle:String — The function handle to be used as a callback after the call is complete or failed.
 
url:String — The URL that is being called.
 
method:String (default = get) — The HTTP method to use, either GET or POST.
 
data:Object (default = null) — The data object to be sent to the server as either GET or POST.
 
contentType:String (default = text/plain) — The content type to send to the server.
